# UI Toolkit Extensions

## Description

UI Toolkit Extensions is a Unity package that enhances the functionality of Unity's UI Toolkit. It provides extended Visual Elements, utilities for creating smooth transitions, and a powerful sprite animator, making it easier to create rich and interactive user interfaces.

## Features

- Extended Visual Elements with additional functionality
- Transition utilities for creating smooth animations
- Sprite Animator for creating complex sprite-based animations
- Easy integration with existing UI Toolkit projects

## Installation

To install this package, follow these steps:

1. Open the Unity Package Manager (Window > Package Manager)
2. Click the '+' button and select "Add package from git URL..."
3. Enter the following URL: `https://github.com/RA-StudioX/UIToolkitExtensions.git`
4. Click 'Add'

## Usage

For detailed usage instructions, please refer to the following documentation:

- [VisualElementExtensions](Documentation~/VisualElementExtensions.md)
- [TransitionExtensions](Documentation~/TransitionExtensions.md)
- [SpriteAnimator](Documentation~/SpriteAnimator.md)

## Requirements

- Unity 2021.3 or later
- UI Toolkit
